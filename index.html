Why React?

  1. Não é um framework MVC.
  2. Não usa templates.
  3. Não exite watch events, podendo renderizar quando for necessário.
  4. Reaproveitamento de código com a criação de componentes
  5. Trabalha com o VirtualDOM, sendo mais rápido que o DOM nativo
  6. Podemos rodar o React no lado do Servidor com Node, melhorando para o SEO e a performace. Veja (apresentação sobre React Isomorfico)
